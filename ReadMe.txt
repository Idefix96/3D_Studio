To get started, you need the latest versions of the following external frameworks:

	Assimp	(Loads various 3D-Fileformats from Hard Disk) 					http://assimp.sourceforge.net/
	Bullet	(Physics Engine) 								http://bulletphysics.org/wordpress/
	Glew	(OpenGL Extensions)								http://glew.sourceforge.net/	
	GLM	(OpenGL Math Library, supplies Vectors, Matrices etc)				https://glm.g-truc.net/0.9.8/index.html
	SFML	(Handles Events such as Keyboard and Mouse input and creates the actual window) https://www.sfml-dev.org/
	SOIL	(loads textures from image files)						http://www.lonesock.net/soil.html

If available, follow the installing instructions from the framework websites. Build the binaries and copy the libraries to your library directory and the 
header files to your include directory
